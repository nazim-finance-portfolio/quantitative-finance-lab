# Quantitative Finance Lab 

A portfolio of financial analytics tools demonstrating skills in **Python, SQL, and Power BI**.

## Projects

### 1. Financial Crime Detection System (New!) 
**Objective:** Analyzed 1.2M+ bank transactions to identify high-risk fraud patterns.
* **Tools:** SQL (Data Extraction), Power BI (Visualization).
* **Key Findings:**
    * Identified **$3.99 Million** in potential fraud losses.
    * Pinpointed **Online Shopping** (`shopping_net`) as the #1 risk category.
    * Isolated **NY & CA** as the highest-risk geographical regions.
* **Visual Evidence:** [View Dashboard Screenshot](Fraud_Analysis_Result.png)

### 2. Algorithmic Trader 
**Objective:** Automated trend-following strategy for AAPL stock.
* **Tools:** Python, Pandas, Plotly.
* **Logic:** Implemented a 50-day Moving Average crossover strategy.

### 3. AI Risk Analyst 
**Objective:** RAG-based AI agent to quantify hidden risk factors.
* **Tools:** LangChain, FAISS, OpenAI.
* **Logic:** Analyzes SEC 10-K filings to extract risk sentiment.
